## Мини-проверка за 15 минут (перед отправкой)

```powershell
cd $env:USERPROFILE\Documents
git clone https://github.com/Ekaterina-Kotendzhi/PPC-Audit-Workspace-v.0.git PPC-Audit-final-check
cd PPC-Audit-final-check
git log -1 --oneline
docker compose up --build
```

1. UI → аудит → Excel → AI → confirm → PDF  
2. `docker compose down`  
3. Вставить ссылки на видео в `ССЫЛКИ-ВИДЕО.md`  
4. `git add` документы → `git commit` → `git push`
